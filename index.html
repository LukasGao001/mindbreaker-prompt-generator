AI 接入（OpenAI 兼容接口，可选）

浏览器直连第三方接口常被跨域(CORS)拦截。推荐使用 OpenRouter（支持浏览器直连，可调用 Kimi / GPT / Claude 等）：
· Base URL：https://openrouter.ai/api/v1
· 模型名示例：moonshotai/kimi-k2、openai/gpt-4o、anthropic/claude-3.5-sonnet
· 到 openrouter.ai 注册并生成 API Key 填到下方。
注意：Kimi/Moonshot 官方地址 api.moonshot.cn 不支持浏览器直连。
说明：Key 只存本机、不上传任何第三方。未启用 AI 时，工具完全本地离线运行。
